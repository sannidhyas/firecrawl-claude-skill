# Versioning

fireclaude follows SemVer.

- **0.x.y** — pre-stable. Breaking changes allowed between any release.
- **1.0.0** — stable public API. Breaking changes only on major bumps.

## Stability checklist before 1.0.0

- [ ] API surface frozen: subcommands, env vars, skill format.
- [ ] Install verified on ≥2 fresh OS (Fedora, Ubuntu, macOS).
- [ ] CI green on ≥3 consecutive runs.
- [ ] README polished, known limits documented honestly.
- [ ] Migration guide from 0.x written.

## Current state (0.6.0)

Pre-stable. Breaking changes are still possible before 1.0.0.
